Running "searchFlights.php"
---------------------------
1. Move all necessary files into the root of Apache or equivalent (ie. htdocs in Apache)
Documents include searchFlights.php, .htaccess, Provider1.txt, Provider2.txt, Provider3.txt

2. Start Apache or equivalent

3. In browser, open up new tab and type in the following :
http://{localhost}/searchFlights/{Origin}/{Destination}

NOTE: If files in seperate folder (ex. searchFlights) then:
http://{localhost}/{folderName}/searchFlights/{Origin}/{Destination}

-----------------------------------------

----------------------------------
